# DOG_BREED_PREDICTION

This project predicts dog breeds from images using transfer learning

---

## Project Files in Repository

- **dog_breed_model.h5** – Trained Keras H5 model (84 MB)  
- **dog_breed_model.keras** – Alternative saved Keras model  
- **show_my_model.py** – Script to load and display the model architecture  
- **app.py** – Main application script to run the project  
- **Demo.mp4** – Project demo video included in the repository  
- **Dog_Breed_Identification_project_doc.docx** – Detailed project document  
- **Dog_Breed_Identification_project_pdf.pdf** – PDF version of the project document  
- **README.md** – This file  
- **static/** – Folder containing CSS and other static files for the web interface  
- **templates/** – Folder containing HTML templates for the web interface  
- **uploads/** – Folder where uploaded images can be placed for prediction  
- **subset/** – Sample training images  

---
